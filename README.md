# agi-number-archer-app
Sample app for #npm agi-number-archer


## Install

1.Clone this repo

> git clone https://github.com/antirek/agi-number-archer-app.git

2.Install deps

> npm install

3.Configure

> nano config.js

4.Run

> node app.js   

//or use pm2 cmd: 

> pm2 start app.js --name "agi-number-archer"
